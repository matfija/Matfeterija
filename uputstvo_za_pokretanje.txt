1. Ukoliko vec niste, instalirate node:
      sudo apt-get install nodejs
2. Sa njim automatski dolazi i node package manager (npm)
3. Proveriti da li su oba instalirana sa:
      node --version
      npm --version
4. Instalirajte MongoDB za vas sistem:
      https://docs.mongodb.com/manual/administration/install-on-linux/
5. Unzipovati direktorijum
6. U terminalu se pozicionirati u server folderu
7. U package.json file-u su dati svi dodatni moduli koji trenutno postoje u
   projektu. Njih instalirate tako sto iz trenutne pozicije pozovete:
      npm install [-g]
   Bez ovog dodatnog -g flaga, dodatni moduli ce se instalirati samo lokalno,
   ali kako cemo mozda potpuno menjati direktorijume i slicno, savetujem da
   instalirate i globalno (svakako se moraju ukljuciti u svaki novi projekat,
   tako da i nije velika razlika zapravo)
8. Pokrenite backend server sa:
      npm start
9. Pokrenite bazu sa:
      mongod --dbpath=/home/teodora/Desktop/matfeterija/server/data --port=3000
   dbpath je putanja do data foldera koji se nalazi u server folderu, tako da
   shodno tome izmenite putanju. Port ostaje isti.
10. Pozicionirajte se u klijent folder.
11. Instalirajte live-server:
      npm install -g live-server
12. Pokrenite live-server iz tekuceg foldera sa:
      live-server
13. Time ste pokrenuli frontend server, i sada mozete videti klijent deo u
    browseru (automatski ce se otvoriti, a ako se ne otvori, u terminalu ce
    vam se ispisati link)
14. Sada mozete koristiti aplikaciju! :D
